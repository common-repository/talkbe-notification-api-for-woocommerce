Talkbe for Woocommerce

== Description ==

This plugin integrates WooCommerce with Talkbe platform.

The plugin allows you to generate rapidly woocommerce api key and set it on Talkbe.
You can find the settings page in WooCommerce tab. The key will be used to register talkbe's webhooks.
For more Talkbe functionality details please refer to its documentation on https://talkbe.app/docs/integrazione-woocommerce-talkbe/
To revoke the key please click on "Revoke key".
